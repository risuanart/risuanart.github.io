# 日宣繪畫室官網

Phase 1：專案骨架＋設計系統地基。目前只有結構與設計系統，**沒有任何頁面內容**，
頁面內容與資訊架構將在 Phase 2 之後補入。

## 專案結構

```
index.html          ← 骨架，不含實際內容
README.md           ← 本檔案
css/
  colors.css         ← 色彩變數（全站唯一色彩來源）
  tokens.css         ← 字級／間距／圓角／陰影／動態 token
  base.css           ← reset 與基礎樣式
js/
  main.js            ← 空白進入點，後續階段的 JS 從這裡開始接
assets/
  images/            ← 空資料夾，圖片素材放這裡
  icons/             ← 空資料夾，圖示素材放這裡
```

技術架構：純 HTML / CSS / JavaScript，部署目標為 GitHub Pages（靜態網站）。
**不使用任何框架**（無 React、Vue、Tailwind、Bootstrap 等），**不引入任何 npm 套件或建置工具**。

## 色彩使用規則

- 所有顏色**必須**來自 `colors.css` 的變數，禁止在任何 HTML／CSS 檔案中硬編碼色碼（如 `#fff`、`rgb(...)`）。
- **一色一責原則**：每個顏色變數只負責一個語意角色（例如「品牌色」「按鈕色」「警示色」），
  不得挪用到角色以外的用途。要新增用途，就新增一個語意明確的變數，不要借用既有變數。
- 目前只有 `--color-brand`（品牌主色，橄欖綠 `#686F12`）已定案，僅用於品牌識別（Logo、識別性強調元素），
  不作為按鈕填色，也不預設為主視覺深底。
- 其餘角色（深底色、按鈕色、警示色、互動色、淺色背景、主要/次要文字色）皆以 TODO 保留在 `colors.css`，
  待使用者拍板後再填入，後續階段不可自行決定色號。

## Token 使用方式

`tokens.css` 提供五類數值系統，元件開發時一律引用這些變數，不要自己另開數值：

- **字級**：`--font-size-xs` 到 `--font-size-2xl` 共 6 級，刻意限制數量以維持排版一致性。
- **間距**：`--space-1` 到 `--space-24`（4/8/16/24/40/64/96px）。原則是**留白刻意不平均**——
  重點區塊周圍留大間距（如 `--space-16`／`--space-24`），相關元素之間用小間距貼近（如 `--space-1`／`--space-2`）。
  禁止全站套用單一均等間距。
- **圓角**：`--radius-sharp`（偏方正）與 `--radius-soft`（偏柔和）兩級，依內容性質選用，不要全站統一套用單一柔和圓角。
- **陰影**：`--shadow-raised` 是雙層組合（`--shadow-contact` 貼邊深色小範圍 ＋ `--shadow-ambient` 大範圍柔和環境陰影），
  需要懸浮感時使用這個組合，不要只套單層模糊陰影（會顯得扁平）。
- **動態**：`--duration-fast/normal/slow` 為節奏階梯；`--ease-standard`／`--ease-overshoot` 為自訂 cubic-bezier，
  不使用瀏覽器預設的 `ease`。動畫節奏要慢、要克制。

`base.css` 已包含 `prefers-reduced-motion` 的全域處理：使用者開啟「減少動態」時，
所有動畫與轉場會被強制縮短，後續加動畫時不需要（也不可以）個別繞過這個設定。

## 設計原則（後續所有階段皆須遵守）

1. **「清楚」永遠優先於炫技**
   動畫與美感是加分項，絕不能犧牲使用者「知道自己在哪」的清楚感與實用性。
   本專案需同時服務企業與幼兒園客戶，資訊必須好找。

2. **語氣依內容性質切分（非全站統一）**

   | 內容性質 | 語氣與視覺調性 |
   |---|---|
   | 空間／質感／視覺氛圍 | 可溫暖、手作感 |
   | 規則說明／須知／老師口吻 | 必須明確直接，不要溫和鋪陳 |

   原因：品牌主理人實際教學風格直接，官網語氣需與現場一致，避免學生因網站語氣溫柔而對到店體驗產生落差。
   視覺也須跟著切分——須知類區塊用清楚條列、明確分隔、少裝飾，不可用柔和模糊的裝飾去包裝規則。

3. **避免 AI 生成的公版感**，明確避免：
   - 米色背景 ＋ 高對比襯線字 ＋ 赤陶橘點綴
   - 近黑背景 ＋ 單一螢光色點綴
   - 全部置中、完全對稱的堆疊式排版
   - 全站統一的柔和圓角與單層模糊陰影
   - 均等分配的留白

4. **不對稱優於對稱**：刻意讓元素偏移、破格，避免「什麼都置中」的安全感。

5. **保留手作的不完美感**：本品牌為手作工作室，適度保留微妙的不規則（如微傾斜角度、不完全工整的圖案），
   比機器般的精確更貼近品牌精神。

6. **動態原則**：
   - 動畫節奏要慢、要克制
   - 進場動畫可用「方向不一致 ＋ 速度錯開 ＋ 逐字」製造個性
   - 過場優先用「元素連續變形」取代硬跳轉，讓使用者能追蹤位置
   - 禁止：彩虹漸層、快速閃爍、大範圍色彩變化、粒子爆炸等公版效果
